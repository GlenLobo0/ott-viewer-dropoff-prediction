# OTT Viewer Drop-off Prediction

This project predicts viewer drop-off and retention patterns for an OTT platform using machine learning models.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Models Used
- Linear Regression
- Random Forest Regressor

## Objective
To analyze user behavior and predict viewer retention and drop-off patterns in Ott platforms (in this we took Netflix as our data).
